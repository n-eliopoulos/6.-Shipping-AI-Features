Module 3: Cost Monitoring

The Cost Reality

Claude Sonnet: ~$3 per million input tokens, ~$15 per million output tokens. That sounds cheap until you do the math at FINRA scale.

A real scenario

A compliance screening feature processes 10,000 documents per day. Each document averages 2,000 input tokens and generates a 500-token analysis.

ComponentDaily tokensDaily costMonthly cost
Input (10K × 2,000)20M tokens$60$1,800
Output (10K × 500)5M tokens$75$2,250
Total25M tokens$135$4,050

That's one feature. Multiply by 5 AI features across the organization and you're looking at $20,000+/month — and that's before anyone makes a mistake with prompt length or forgets to set a budget cap.
